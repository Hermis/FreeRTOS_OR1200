FreeRTOS_OR1200
===============
The FreeRTOS support OpenRISC and Verify for FPGA Board.

FPGA Board:

1.SMIMS Marcube(Xilinx XC6SLX75T)

2.De0_nano(Altera)

//------------------------------------------------------------

Project Step:

1.cd FreeRTOS_OpenRISC/Demo/OpenRISC_OR1200_GCC/

2.Check your CCPATH(or1k Toolchain PATH) on Makefile

3.make
